## Recursos de R

### Tutoriales y Cursos
- [R for Data Science](https://r4ds.had.co.nz/) - Libro gratuito de Hadley Wickham sobre ciencia de datos con R
- [Advanced R](https://adv-r.hadley.nz/) - Libro avanzado de programación en R
- [R Programming for Data Science](https://bookdown.org/rdpeng/rprogdatascience/) - Libro de Roger Peng
- [Swirl](https://swirlstats.com/) - Aprende R de forma interactiva dentro de R
- [R Basics - Quick Reference](https://www.rstudio.com/resources/cheatsheets/) - Hojas de referencia rápida de RStudio

### Bioinformática con R
- [Bioconductor](https://www.bioconductor.org/) - Proyecto de software de código abierto para análisis de datos genómicos
- [R for Biologists](https://www.rforbiologists.org/) - Recursos específicos para biólogos
- [Data Analysis for the Life Sciences](https://rafalab.github.io/pages/harvardx.html) - Curso de análisis de datos de Harvard

### Visualización
- [ggplot2 Documentation](https://ggplot2.tidyverse.org/) - Documentación oficial de ggplot2
- [R Graphics Cookbook](https://r-graphics.org/) - Libro sobre gráficos en R
- [Fundamentals of Data Visualization](https://clauswilke.com/dataviz/) - Principios de visualización de datos

## R con GitHub

### Integración de R y Git
- [Happy Git and GitHub for the useR](https://happygitwithr.com/) - Guía completa de Jenny Bryan para usar Git con R
- [usethis package](https://usethis.r-lib.org/) - Paquete para automatizar tareas de desarrollo en R
- [Version Control with Git and SVN](https://support.rstudio.com/hc/en-us/articles/200532077-Version-Control-with-Git-and-SVN) - Documentación de RStudio sobre control de versiones

### Paquetes de R en GitHub
- [R Packages](https://r-pkgs.org/) - Libro de Hadley Wickham sobre desarrollo de paquetes
- [GitHub Actions for R](https://github.com/r-lib/actions) - Integración continua para proyectos R
- [pkgdown](https://pkgdown.r-lib.org/) - Sitios web para paquetes de R

## Bash y Terminal

### Tutoriales Básicos
- [The Unix Shell](https://swcarpentry.github.io/shell-novice/) - Tutorial de Software Carpentry
- [Linux Command Line Basics](https://ubuntu.com/tutorials/command-line-for-beginners) - Tutorial de Ubuntu
- [Bash Guide for Beginners](https://tldp.org/LDP/Bash-Beginners-Guide/html/) - Guía completa de Bash
- [ExplainShell](https://explainshell.com/) - Herramienta para explicar comandos de shell

### Scripting
- [Advanced Bash-Scripting Guide](https://tldp.org/LDP/abs/html/) - Guía avanzada de scripting
- [Bash Scripting Tutorial](https://linuxconfig.org/bash-scripting-tutorial-for-beginners) - Tutorial para principiantes
- [ShellCheck](https://www.shellcheck.net/) - Herramienta para verificar scripts de shell

### Bioinformática en la Terminal
- [Bioinformatics Data Skills](https://vincebuffalo.com/book/) - Libro de Vince Buffalo
- [Unix for Bioinformatics](https://bioinformaticsworkbook.org/Appendix/Unix/unix-basics-1.html) - Conceptos básicos de Unix para bioinformática
